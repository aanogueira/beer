# Beer: Contador de Bebidas

This is a simple application that allows you to keep track of the number of drinks you have consumed. It is built using Next.js and TypeScript.
It provides a user-friendly interface that allows adding teams and updating the count of drinks consumed by each team.

It's a pretty simple application. The record are store in browser local storage, which means that the data will be lost if you clear your browser's local storage or if you switch to a different browser or device.
For the purpose of this project, is no big deal, as it's just a simpler way to store the data without having to set up a backend server.
